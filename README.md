# Axiom Token Table (Mock)

Pixel-approx build of Axiom Trade Pulse token table with realtime updates, sorting, popover, tooltip, modal, skeleton/shimmer, and simple error boundary.

## Stack
- Next.js 14 App Router + TypeScript (strict)
- Tailwind CSS
- Redux Toolkit (UI state: sort, pin)
- React Query (data fetching)
- Radix UI primitives (Tooltip, Popover, Dialog)

## Run
```bash
npm i   # or yarn / pnpm
npm run dev
# open http://localhost:3000
```

## Deploy
- Push to GitHub, import on Vercel (Node 18+)
